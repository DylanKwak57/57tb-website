## 1. Overview

### 1.1 Design Goals

- кі к°қмқҙ мҠӨнҒ¬лЎӨн•ҳлҠ” кІғл§ҢмңјлЎң "м—¬кё°м„ң лЁёлҰ¬н•ҳкі  мӢ¶лӢӨ"лҠ” к°җм •мқ„ лҠҗлҒјлҠ” лӘ°мһ…нҳ• кІҪн—ҳ
- л°©мҪ• н•ңкөӯ лҜёмҡ©мӢӨ мӨ‘ м••лҸ„м ҒмңјлЎң м°Ёлі„нҷ”лҗң н”„лҰ¬лҜём—„ мӣ№мӮ¬мқҙнҠё
- лӘЁл°”мқј нҚјмҠӨнҠё + 3к°ң көӯм–ҙ(TH/EN/KO) мҷ„м „ м§Җмӣҗ
- LINE мҳҲм•Ҫ м—°лҸҷ (кё°мЎҙ мһҗмІҙ мҳҲм•Ҫ мӢңмҠӨн…ң: 57totalbeauty.net/booking)

### 1.2 Design Principles

- **Cinematic First**: лӘЁл“  м„№м…ҳмқҙ мҳҒнҷ”мқҳ н•ң мһҘл©ҙмІҳлҹј лҠҗк»ҙм ём•ј н•Ё
- **Mobile First**: нғңкөӯ мӮ¬мҡ©мһҗ 70%+ лӘЁл°”мқј вҶ’ лӘЁл°”мқјл¶Җн„° м„Өкі„
- **Performance**: нғңкөӯ л„ӨнҠёмӣҢнҒ¬ нҷҳкІҪ кі л Ө, LCP < 2.5s
- **Simplicity**: м •ліҙ кіјл¶Җн•ҳ м—Ҷмқҙ, к°җм„ұ вҶ’ н–үлҸҷ(мҳҲм•Ҫ) мң лҸ„

## 4. UI/UX Design

### 4.1 Design System

#### Color Palette

```
// globals.css @theme (Tailwind v4 CSS Variables)
// Dark Mode (default)
colors: {
  brand: {
    black:     '#0A0A0A',   // Primary background
    dark:      '#111111',   // Section background
    card:      '#1A1A1A',   // Card background
    gold:      '#C9A96E',   // Accent, CTA, highlights
    champagne: '#E8D5B7',   // Secondary accent
    white:     '#FFFFFF',   // Primary text
    gray:      '#6B7280',   // Muted text
    'gray-light': '#9CA3AF', // Borders, dividers
  }
}

// Light Mode ([data-theme="light"] override)
colors: {
  brand: {
    black:     '#FAFAFA',   // Light background
    dark:      '#F0F0F0',   // Light section background
    card:      '#FFFFFF',   // White card background
    gold:      '#A08030',   // Darker gold for contrast
    champagne: '#6B5A3F',   // Darker champagne for readability
    white:     '#1A1A1A',   // Dark text
    gray:      '#6B7280',   // Same muted text
    'gray-light': '#4B5563', // Darker for readability on light
  }
}
```

#### Theme Toggle

- iOS-style slide switch in Header (gold knob with sun/moon icons)
- State managed via `ThemeContext` + `localStorage` persistence (key: `57tb-theme`)
- Flash prevention via inline script in `<head>` before React hydration
- Default: Dark mode

## 8. Coding Conventions

### 8.1 Naming Conventions

| Target | Rule | Example |
|--------|------|---------|
| Components | PascalCase | `HeroSection`, `ServiceCard` |
| Functions | camelCase | `formatPrice()`, `handleScroll()` |
| Constants | UPPER_SNAKE_CASE | `SERVICES_DATA`, `BRANCHES` |
| Types | PascalCase | `Service`, `GalleryItem` |
| Files (component) | PascalCase.tsx | `HeroSection.tsx` |
| Files (utility) | camelCase.ts | `utils.ts` |
| Folders | kebab-case | `before-after/` |
| CSS classes | Tailwind utility-first | `className="flex items-center"` |
