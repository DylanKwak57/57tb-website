## 4. UI/UX Design

### 4.1 Design System

#### Color Palette

```
// globals.css @theme (Tailwind v4 CSS Variables)
// Dark Mode (default)
colors: {
  brand: {
    black:     '#0A0A0A',   // Primary background
    dark:      '#111111',   // Section background
    card:      '#1A1A1A',   // Card background
    gold:      '#C9A96E',   // Accent, CTA, highlights
    champagne: '#E8D5B7',   // Secondary accent
    white:     '#FFFFFF',   // Primary text
    gray:      '#6B7280',   // Muted text
    'gray-light': '#9CA3AF', // Borders, dividers
  }
}

// Light Mode ([data-theme="light"] override)
colors: {
  brand: {
    black:     '#FAFAFA',   // Light background
    dark:      '#F0F0F0',   // Light section background
    card:      '#FFFFFF',   // White card background
    gold:      '#A08030',   // Darker gold for contrast
    champagne: '#6B5A3F',   // Darker champagne for readability
    white:     '#1A1A1A',   // Dark text
    gray:      '#6B7280',   // Same muted text
    'gray-light': '#4B5563', // Darker for readability on light
  }
}
```

#### Theme Toggle

- iOS-style slide switch in Header (gold knob with sun/moon icons)
- State managed via `ThemeContext` + `localStorage` persistence (key: `57tb-theme`)
- Flash prevention via inline script in `<head>` before React hydration
- Default: Dark mode
